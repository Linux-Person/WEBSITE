13 Apr 2026
Updates server to project poseidon version 1.0.2
The MOTD ping works by intercepting the 0xFE ping packet before normal login handling, sending back the MOTD, current player count and max players, then closing the connection. This extra connection and response is what can occasionally cause end of stream errors when connecting shortly after pinging the server list with Beta Tweaks. If you experience connection issues with Beta Tweaks server ping enabled, either wait around 30 seconds after the server list refreshes before connecting, or disable server ping in Beta Tweaks entirely. This feature is purely cosmetic and completely optional — the server works fine without it. Disabling server ping in Beta Tweaks will prevent any related connection issues entirely. if this causes issues we will most likely remove this feature but most likely not because you can just easily disable it in beta tweaks and most people probably don't use jar mods but that is just a guess.
